## Batch-Tamarin

If you are using Tamarin from the command line, want to analyze many properties, or use large computing clusters, then you might want to check out [<code>batch-tamarin</code>](/2025/10/30/batch-tamarin-now-available.html). <code>batch-tamarin</code> offers many helpful features, including resource management, caching, attacks/counterexample output, improved reproducibility, and much more.

 - [Blog post](/2025/10/30/batch-tamarin-now-available.html) with an overview of features 
 - [Source code](https://github.com/tamarin-prover/batch-tamarin)
